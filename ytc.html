YouTubeCacher

YouTubeCacher is a GUI frontend for Windows, written in C and using the Windows API, to control yt-dlp and compatible tools.

YouTubeCacher is licensed under the MIT License.

YouTubeCacher has been known to run on 32-bit and 64-bit versions of x86 and 64-bit ARM.

You can find the YouTubeCacher source code in its GitHub repository.
